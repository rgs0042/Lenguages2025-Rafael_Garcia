Resumen de XML schema

En este documento se puede encontrar los principales conceptos de XSD. Siendo XSD un una forma más mejorada de DTD que le da forma nuestros docuemtos XML.

Validaciones con XSD

    Documento con las reglas. Es el esquema XML

  • es un archivo XML.
  • Usa espacio de nombre.
  • Su extensión es .XSD

    Documento XML que se valida.

  • Insatancia del esquema.

    El propósito del esquema XML es definir:

  • Los elementos y atributos que pueden aparecer en un documento.
  • El número y orden elementos secundarios.
  • Los tipos de datos para elementos y atributos.
  • Los valores predeterminados y fijos para elementos y atributos

Está compuesto por:

  • Definiciones
  • Raíz
  • Elementos
  • Atributos
  • Tipos

El componente xs:element se diferencia en tipos de datos:

  • String
  • Boolean
  • Decimal
  • Integer
  • PositiveInteger
  • NegativeInteger
  • Float
  • Double
  • DateTime
  • Date
  • Time

El elemento se define una vez, se usa el atributo ref, esto es solo para organización del documento
